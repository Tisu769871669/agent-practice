# Agent Practice

Agent Practice is an open-source practice platform for learning AI agent engineering through graded, local challenges.

The first release is designed as a static website plus a GitHub repository. Learners browse a LangChain/LangGraph-oriented path on the website, clone the repo, solve challenges locally with any framework, and run a deterministic runner/grader to get feedback.

## MVP Scope

- Static website for the learning path, challenge catalog, challenge details, and contribution guide.
- 30-challenge roadmap from foundations to expert agent systems.
- 10 runnable challenges in the first release.
- Local Python runner that writes `result.json` and `transcript.jsonl`.
- Starter templates for raw Python, LangChain, and LangGraph.
- Framework-neutral challenge interface.
